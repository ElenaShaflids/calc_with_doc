Simple Console Calculator
=========================

.. module:: simple_calculator
   :platform: any
   :synopsis: A simple console calculator for performing basic mathematical operations.

add(x, y)
-----------

.. function:: add(x, y)

   Adds two numbers.

   :param x: First number.
   :param y: Second number.
   :return: Sum of numbers x and y.

subtract(x, y)
----------------

.. function:: subtract(x, y)

   Subtracts one number from another.

   :param x: First number.
   :param y: Second number.
   :return: Difference of numbers x and y.

multiply(x, y)
----------------

.. function:: multiply(x, y)

   Multiplies two numbers.

   :param x: First number.
   :param y: Second number.
   :return: Product of numbers x and y.

divide(x, y)
-------------

.. function:: divide(x, y)

   Divides one number by another.

   :param x: Dividend.
   :param y: Divisor.
   :return: The result of dividing x by y, or an error message when dividing by zero.
